Mini tutoriel Angular

Angular est un framework qui permet d'écrire des sites web, ce qui veut dire le front-end. Angular utilise principalement le langage TypeScript. Ce framework a été développé par Google et il est open-source. Un projet typique angular est composé de composants, de services et de metadata.

Angular is a framework used to make websites. Angular is an open-source TypeScript based framework that was created by Google. A typical Angular project is made up of components and services.

Mini tutoriel Crud

Crud est un acronym pour Create Read Update Delete. Le Crud est les fonctions nécessaires pour gérer des données stockées.

Crud is an acronym for Create Read Update Delete. Crud refers to the four methods used when dealing with stored data.

Mini tutoriel Relations

Il y a beaucoup de types de relation différents. Il y a l'héritage, l'implémentation, l'association, la composition et l'agrégation. Chacun de ces relations sont représentées par une ligne ou une flèche différente. La composition permet d'indiquer une classe ou un objet qui ne peut pas exister sans l'autre.
